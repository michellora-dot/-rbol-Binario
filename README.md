# Árbol Binario en Java

## ¿Qué es un árbol binario?
Un árbol binario es una estructura de datos donde cada nodo tiene como máximo dos hijos: izquierdo y derecho. Se utiliza para organizar datos de forma jerárquica y facilitar búsquedas.

## ¿Cómo se implementó?
Se creó una clase Nodo para representar cada elemento del árbol y una clase ArbolBinario que permite:

- Insertar datos
- Recorrer el árbol en inorden
- Buscar un número

Se utilizó un menú en consola para interactuar con el usuario.

## Ejemplo de ejecución
--- MENU ---

1. Insertar número
2. Mostrar recorrido inorden
3. Buscar número
4. Salir

Ingrese número: 10
Ingrese número: 5
Ingrese número: 15

Recorrido inorden:
5 10 15

Buscar número: 10
El número SI existe
